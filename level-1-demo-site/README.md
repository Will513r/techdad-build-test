# Level 1 — Demo Site

Build the kind of page we send a prospect to win their business: **one self-contained landing page** that looks sharp, loads instantly, and is built from the prospect's real info. This is our outreach hook, so it has to feel custom — not like a template.

**Plan for ~2–3 hours.** Clean and convincing beats big and broken.

---

## The task

One file: **`index.html`**. Vanilla HTML, CSS, and JavaScript — no frameworks, no build step, no external dependencies you can avoid. It must open and work by **double-clicking the file** (no server needed).

We're looking for:

1. **Real business data throughout** — the info below, used naturally. No filler.
2. **A single, self-contained file** — CSS in a `<style>`, JS in a `<script>`, all in `index.html`.
3. **One signature element** — something memorable that fits a cleaning company (a before/after slider, an animated water effect, a clean custom badge — your call). This is what separates a demo that sells from a generic brochure.
4. **SEO foundation** — exactly one `<h1>`, a keyword-forward `<title>` and meta description, `LocalBusiness` JSON-LD, and `FAQPage` JSON-LD from the FAQs.
5. **Mobile-first** — looks right at 375px, 768px, and 1024px+.

### Suggested sections (in this order)

1. Hero — full height, headline from the best selling point, one clear call-to-action
2. Trust strip — years, rating, areas served
3. Services — a card per service
4. Gallery — placeholder/stock images are fine (note the source in a comment)
5. Reviews — use the two below
6. Contact — an estimate form that validates and shows a success state (no backend needed)
7. Footer — include a "Demo site built by Tech Dad Media" credit

> Demo-stage shortcuts are fine and expected: placeholder/stock images (flagged in a comment) and a frontend-only form. We're selling the vision here, not shipping production.
